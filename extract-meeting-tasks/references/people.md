# Known People Reference

Speech-to-text models frequently misspell names, especially non-English ones. This file maps known people to their correct names, common transcription errors, and project field values.

Use this file alongside the Granola participant list (which includes real names and emails) to correctly identify people in transcripts.

## How to use

1. Read the Granola participant list from meeting metadata (names + emails)
2. Read the transcript and identify all names mentioned
3. Match transcript names against this file and the participant list
4. Flag any unresolved names for user confirmation before extraction

## Adding New People

When a name appears in a transcript that doesn't match this file or the participant list, flag it during the confirmation step. After the user confirms the correct name and any known misspellings, add them here for future meetings.
